# Resume Description

Designed a reusable open-source MLOps platform to deploy any ML model through a full production lifecycle using DVC, lakeFS, Great Expectations, Feast, MLflow Registry, Kubeflow, Katib, BentoML, Docker, Kubernetes, KServe, HPA, Evidently, NannyML-style alerts, Prometheus/Grafana, Helm and Argo CD.

# Interview Pitch

This project is a model-agnostic MLOps platform. Any model can be plugged in through a standard adapter interface. The system handles data versioning, validation, feature management, training, experiment tracking, model registry, packaging, Kubernetes deployment, KServe inference, autoscaling, drift monitoring, retraining triggers and GitOps release. The goal is to demonstrate the complete production ML lifecycle using open-source tools.
